Understanding AB2D Data

The AB2D API provides stand-alone Prescription Drug Plan (PDP) sponsors with claims data that can be used to enhance the use of medications and improve the long term health of beneficiaries. CMS has created some resources to help PDP sponsors understand and utilize this data - check out sample claim files, the AB2D data dictionary, and more.
