# AB2D BFD

This library implements the connection and configuration to BFD for AB2D.

## Using the Library

Add to Maven Project

```
<dependency>
    <groupId>gov.cms.ab2d</groupId>
    <artifactId>ab2d-bfd</artifactId>
    <version>1.0</version>
</dependency>
```

Adding to Gradle

```
implementation "gov.cms.ab2d:ab2d-bfd:1.0"
```
